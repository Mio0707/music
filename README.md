# 儿童音乐学习产品研究

本仓库用于保存儿童音乐学习产品的课程标准、竞品研究和交互截图，方便在不同电脑之间继续使用 Codex 协作。

## 材料目录

- `materials/义务教育艺术课程标准（2022年版）.pdf`：课程标准原始文件。
- `research/课程标准音乐部分学习摘要.md`：音乐课程目标、学习任务和产品启示。
- `research/音乐交互产品研究摘要.md`：Incredibox、Groove Pizza、Variation Playground 与 Chrome Music Lab Rhythm 的交互分析。
- `output/incredibox-audit/`：产品实测截图。

## 说明

`tmp/` 是 PDF 页面渲染缓存，可以从原始 PDF 重新生成，因此不提交到仓库。
