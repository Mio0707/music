# 儿童音乐学习产品

这个项目帮助6—12岁儿童先听懂音乐，再用拍手、动物乐队和AI支持完成自己的作品。

## 只看这两份文件

- [PRODUCT.md](PRODUCT.md)：唯一的产品方案，包含教学、创作、动物乐队、AI边界和Demo规则。
- `README.md`：项目入口、当前状态和使用方法。

历史讨论、零散理论和旧任务表已经关闭，不再作为产品依据。

## 当前成果

- `prototype/index.html`：可点击核心流程原型。
- `prototype/app.js`、`prototype/styles.css`：原型交互与视觉。
- `prototype/assets/stickers/`：五只动物的头像和演奏贴纸。
- `materials/义务教育艺术课程标准（2022年版）.pdf`：课程标准原件。
- `output/`：竞品实测截图。

## 打开原型

直接双击 `prototype/index.html`。

如需测试麦克风，在项目目录运行：

```powershell
python -m http.server 4173
```

然后打开 `http://127.0.0.1:4173/prototype/`。
