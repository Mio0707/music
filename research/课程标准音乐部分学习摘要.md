# 《义务教育艺术课程标准（2022年版）》音乐部分学习摘要

## 核心素养

艺术课程围绕四项核心素养展开：

1. 审美感知：发现、感受和理解音乐的声音特征与美感。
2. 艺术表现：通过演唱、演奏、律动等方式表达音乐。
3. 创意实践：运用声音、节奏、旋律和技术工具进行创造。
4. 文化理解：理解音乐与生活、地域、民族和文化环境的关系。

## 音乐学习实践

音乐学习可以概括为四类活动：欣赏、表现、创造与联系。产品不应只训练操作技巧，还要让儿童经历聆听、表达、创作、分享和反思。

## 创作学习闭环

适合转化为产品流程的创作过程是：

> 产生动机 → 构思 → 创作与制作 → 试唱、试奏或表演 → 展示 → 交流评价 → 修改完善

AI适合承担提示、伴奏、示范和反馈角色，但儿童仍应是作品的作者和最终选择者。

## 对儿童音乐产品的启示

- 重点年龄可放在三至五年级，并兼顾6—12岁儿童。
- 从生活声音、自然声音和身体节奏进入，比先讲抽象乐理更自然。
- 每一次操作都应同时提供可听、可见、可模仿的反馈。
- 用比较帮助儿童理解音乐变化，例如更快、更慢、更强、更弱、明亮、安静。
- 创作结束后应保留展示、讲述和再次修改的机会。
- 现代技术和人工智能可以支持创作，但不能替儿童完成全部表达。
