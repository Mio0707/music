# 儿童音乐交互产品研究摘要

## 总体结论

四款产品分别把抽象音乐概念转化成儿童容易操作的视觉对象：

| 产品 | 核心比喻 | 教学优势 |
| --- | --- | --- |
| Incredibox | 把音轨变成人物 | 自由混音、表达和表演 |
| Groove Pizza | 把节奏变成几何图形 | 理解拍点、循环和数学关系 |
| Variation Playground | 把作曲参数变成卡片 | 理解同一素材如何产生变化 |
| Chrome Music Lab Rhythm | 把节拍变成会动的角色 | 低门槛节奏启蒙 |

## Incredibox

儿童先选择音乐风格，再把声音图标拖到人物身上。人物会开始演唱或演奏，成为一个循环播放的声音层。继续添加人物即可形成多轨混音，特定组合还会解锁动画奖励。

- 人物代表正在播放的声音层。
- 图标代表预制声音循环，而不是单个音符。
- 服装和动作帮助儿童识别声音来源与播放状态。
- 优点是几乎不会产生明显错误的组合，能鼓励大胆尝试。
- 局限是音乐概念解释较弱，需要教师或引导角色补充“为什么好听”。

官方资料：<https://www.incredibox.com/info/faq>

## Groove Pizza

圆形代表一小节，同心圆代表不同打击乐器层，圆周位置表示声音在循环中出现的时间。

| 视觉元素 | 音乐含义 |
| --- | --- |
| 三角形 3 | 一小节内均匀出现3次声音 |
| 正方形 4 | 均匀出现4次声音 |
| 五边形 5 | 均匀出现5次声音 |
| 六边形 6 | 均匀出现6次声音 |
| 八边形 8 | 均匀出现8次声音 |
| 0 | 清空当前节奏层 |
| 旋转形状 | 改变节奏的开始位置 |

形状的顶点数量代表声音出现次数，而不是乐器或音高。圆形适合呈现循环，下方线性网格适合精确编辑。

官方资料：<https://www.musedlab.org/groovepizza/>

## Variation Playground

Britten卡片版本先选择乐器，再从视觉卡片中选择音乐素材和变化参数。

| 形状 | 含义 |
| --- | --- |
| 三角形 | Triad，三和弦素材 |
| 半圆或圆拱 | Moving Scale，流动音阶 |
| 向下斜坡 | Pattern Down，向下进行的音型 |
| 五边形 | Ending，结束句 |
| 翻转 | Inversion，改变和弦转位或音型方向 |

其他视觉参数包括：暖色表示大调，蓝色表示小调；`p / mf / f` 表示力度；Largo、Moderato、Presto表示速度；平滑边缘和锯齿边缘表示没有装饰或加入装饰。

官方资料：<https://www.musedlab.org/variation-playground/>

## Chrome Music Lab Rhythm

儿童点击小灰点放入声音，播放时角色同步敲击对应乐器。

- 横向位置代表声音出现的时间。
- 不同纵向行代表不同乐器。
- 小灰点表示关闭，大的彩色形状表示已经放入声音。
- 形状与颜色用于区分乐器层，不表示音高。
- 角色动作把“谁在什么时候发声”直接演示出来。

它最适合低龄节奏启蒙，因为反馈链非常短：点击一个点、听见声音、看见角色动作、理解时间位置。

官方体验：<https://musiclab.chromeexperiments.com/Rhythm/>

## 对目标产品的组合建议

1. 用 Rhythm 的点击节拍作为最低门槛入口。
2. 用 Groove Pizza 的圆环呈现循环和节奏形状。
3. 用 Incredibox 的角色承载环境声音、打击乐和哼唱等音轨。
4. 用 Variation Playground 的卡片提供速度、力度、情绪和装饰变化。
5. 让儿童进行前后对比，最后录制、展示、讲述并修改作品。

适合的核心原则是：每一种声音都有一个看得见、能触摸、会回应的身体。

## 无障碍注意事项

- 不要只靠颜色传达含义，同时使用形状、文字和声音提示。
- 同时支持点击与拖拽，并提供较大的触控区域。
- 提供暂停动画、减少动态效果和清晰焦点状态。
- 对关键图标增加可读标签，让键盘和辅助技术也能操作。
