# 可点击原型

这是“动物音乐明信片”的第一版核心流程原型，不需要安装项目依赖。

## 最简单的打开方式

双击 `index.html`，即可体验：

选择情绪 → 4 拍节奏热身 → 4 拍屏幕轻拍录制 → 动物编排 → AI 版本对比 → 音乐明信片。

当前 Demo 只使用四分音符位置，即每拍最多记录一次拍手；后续阶段会在相同的 4 拍内加入八分音符位置。

直接打开文件时，可以点击中间的拍手按钮模拟拍手。浏览器可能不会允许网页使用麦克风，这是浏览器的安全限制，不影响其他流程。

## 需要测试麦克风时

在本目录打开终端，运行：

```powershell
python -m http.server 4173
```

然后在浏览器访问：

```text
http://127.0.0.1:4173
```

浏览器询问麦克风权限时选择允许。测试结束后，在终端按 `Ctrl+C` 关闭预览服务。

## 当前原型说明

- 声音由浏览器实时合成，用于验证交互，不是最终音乐素材。
- “AI 润色”用不同的合成变化模拟，还没有连接在线模型。
- 保存功能使用当前浏览器的本地存储。
- 分享只显示成年人确认流程，不会真的上传或发送作品。
- 动物头像使用透明贴画素材；编排页上方常驻纸偶舞台，点击试听后当前乐段的动物从舞台下方翻转上场，同时可继续查看和编辑下方时间轴。
